# Daily Health Tracker

A single-page habit tracker built around a Mediterranean diet and progressive fitness plan.

## Features

- **Phase-aware checklist** — items appear based on your current program week
- **Expandable details** — tap any item for quick-reference tips
- **Progress tracking** — daily completion bar with streak counter
- **History calendar** — month view with color-coded days
- **Reference docs** — built-in viewer for quick reference, mix-and-match meals, and shopping list
- **Multi-profile** — multiple users on the same device
- **Export/import** — back up and restore your data
- **100% private** — all data stays in localStorage, nothing is sent anywhere

## Getting Started

1. Open `index.html` in any browser
2. Enter your name
3. Set your program start date
4. Check off items as you complete them each day

## Program Structure

| Week | New Items Added |
|------|----------------|
| 1 | Meditation, exercise (walking), diet, supplements |
| 3 | Core + hip exercises (Routine A) |
| 4 | Resistance training (Routine B) |

## No Server Required

Single HTML file — no build tools, no dependencies, no accounts.
